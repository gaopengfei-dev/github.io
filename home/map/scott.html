# 高德地图

# 简介

高德地图 JS API 是一套 JavaScript 语言开发的的地图应用编程接口，移动端、PC 端一体化设计，一套 API 兼容众多系统平台。目前 JS API 免费开放使用

JS API 提供了 2D、3D 地图模式，满足绝大多数开发者对地图展示、地图自定义、图层加载、点标记添加、矢量图形绘制的需求，同时也提供了 POI 搜索、路线规划、地理编码、行政区查询、定位等众多开放服务接口。

# 准备

1、登录高德地图 api 注册账号 成为开发者 点击注册

2、登录后进入应用管理、创建新应用

3、为应用添加 key，获取 key 值（服务平台一项选择 web 端 jsApi）

# 在 Vue 中的使用方法

1、复制上一步获取的 key 将以下代码添加到脚手架中 public 下的 index.html 中。

<script
  type="text/javascript"
  src="https://webapi.amap.com/maps?v=1.4.15&key=您申请的key值"
></script>

# 演示

演示代码


<template>
  <div id="container"></div>
</template>

<script>
export default {
  data() {
    return {
      map: null
    };
  },
  mounted() {
    this.getAMap();
  },
  methods: {
    getAMap() {
      this.map = new AMap.Map('container', {
        zoom: 11, //级别
        center: [117.132461, 39.222528], //中心点坐标
        viewMode: '3D', //使用3D视图
        mapStyle: 'amap://styles/whitesmoke' // 自定义样式
      });
    }
  }
};
</script>
