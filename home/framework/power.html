# 权限管理系统

权限管理系统脚手架也是一套前端后台管理系统基础模版，具有易迁移性的特点，操作简单方便，前端开发可以在此脚手架的基础上进行开发，也可以后迁移此脚手架到项目中。

注

使用此脚手架的前提条件：后端代码也要使用公司后端权限管理系统脚手架。否则逻辑、接口无法对应。

# 文件结构

特殊文件解释:

--directive 自定义指令文件夹，存放检验按钮权限的自定义指令。代码如下：

import Vue from 'vue';
import store from '../store';
// 自定义指令按钮权限
Vue.directive('has', {
  inserted: function(el, binding) {
    if (!permissionJudge(binding.value)) {
      el.parentNode.removeChild(el);
    }
    function permissionJudge(value) {
      let list = store.state.user.userInfo.buttonList;
      for (let item of list) {
        if (item.url === value) {
          return true;
        }
      }
      return false;
    }
  }
});

路由权限配置方法

菜单级路由,在 router/index.js 中

//  为一级二级菜单加路由权限
let menuList = localStorage.getItem('userInfo')
  ? JSON.parse(localStorage.getItem('userInfo')).menuList
  : [];
let routesChildren = routes[0].children.map((v) => v.path);
function getRoutePower(menuList) {
  menuList.forEach((v) => {
    if (v.children.length) {
      getRoutePower(v.children);
    } else {
      if (routesChildren.indexOf(v.url) !== -1) {
        routes[0].children[routesChildren.indexOf(v.url)].meta.power = true;
      }
    }
  });
}
getRoutePower(menuList);

// 路由拦截中配置
router.beforeEach((to, from, next) => {
  Vue.prototype.NProgress.start();
  // 判断如果是从登录页面来的则调用处理方法
  if (from.path === '/login') {
    let menuList = localStorage.getItem('userInfo')
      ? JSON.parse(localStorage.getItem('userInfo')).menuList
      : [];
    getRoutePower(menuList);
  }
  const token = store.state.user.token;
  if (token) {
    //  判断routesChildren中是否有将要前往的菜单路由
    if (routesChildren.indexOf(to.path.slice(1)) !== -1) {
      // 判断meta标签中的power属性，为true，有权限，否则无权限
      if (to.meta.power) {
        next();
      } else {
        next({ path: '/404' });
      }
      // 没有则为全局路由，不做拦截
    } else {
      next();
    }
  } else {
    if (to.path === '/login') {
      next();
    } else {
      next({ path: '/login' });
    }
    Vue.prototype.NProgress.done();
  }
});

-- utils 中的 permissionJudge 鉴权方法。

使用场景：

1、在表格操作栏中，若操作栏中的按钮均无权限，则此操作栏没必要显示。可使用 v-if 配合此方法。

2、用于按钮级路由鉴权

// 按钮级权限的第二种方法 也是控制按钮级路由权限的方法
export const permissionJudge = (value) => {
  let list = store.state.user.userInfo.buttonList;
  if (!list) return true;
  for (let item of list) {
    if (item.url === value) {
      return true;
    }
  }
  return false;
};
